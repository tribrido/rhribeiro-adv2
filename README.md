# Rafael Ribeiro Advocacia — Landing Page

Landing page institucional para apresentação do advogado Rafael Henrique Ribeiro, com foco em Direito Cível e Imobiliário.
O projeto é uma página estática, leve e responsiva, com informações de apresentação e canais de contato.

## Visão geral

A página foi construída para ser simples, rápida e compatível com dispositivos móveis.
Ela reúne:

- **Seção hero** com apresentação profissional e identidade visual.
- **Canais de contato** com links diretos para e-mail, WhatsApp, LinkedIn e Instagram.
- **Rodapé** com aviso de uso e direitos reservados.

## Tecnologias

- **HTML5** (estrutura semântica)
- **CSS3** (estilos base e responsividade)
- **JavaScript** (comportamentos simples)

## Estrutura do projeto

```
.
├── Imagens/          # Logos, fotos e imagens de fundo
├── css/              # Estilos principais e responsivos
├── js/               # Scripts do site
└── index.html        # Página principal
```

## Como executar localmente

Como é um site estático, basta abrir o arquivo `index.html` no navegador.
Se preferir, você pode subir um servidor local simples:

```bash
# Python 3
python -m http.server 8000
```

Depois acesse: `http://localhost:8000`

## Boas práticas adotadas

- **Semântica HTML** para melhor acessibilidade e SEO.
- **Links externos com `rel="noopener"`** para segurança.
- **Layout responsivo** com CSS dedicado.
